### Simple showcase of Selenium and BeautifulSoup usage in Python (using Firefox as the browser of choice).

example_com loads [example.com](https://www.example.com) and showcases:
* Scraping text from webpages
* Clicking links to new pages
* Waiting for those pages to load
* Entering text
* Using Beautiful Soup to parse more complex page elements

tab_switching loads the above and [my GitHub](https://github.com/mpvio) and showcases:
* Running Javascript code
* Using window handles to track tabs
* Tab switching
* Scraping information from multiple tabs
* Closing specific tabs while keeping the driver running
